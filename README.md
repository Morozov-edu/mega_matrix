# MegaMatrix

MegaMatrix — это библиотека на Ruby для работы с матрицами.  
Проект реализует базовые операции линейной алгебры и удобный интерфейс для работы с матричными данными.

---

## 🚀 Возможности

- Создание матриц
- Сложение и вычитание матриц
- Умножение на скаляр
- Матричное умножение
- Доступ к элементам
- Красивый вывод матриц
- Расширяемая архитектура (разделение по модулям)

---

## 📦 Установка

### 1. Сборка gem

```bash
gem build mega_matrix.gemspec
2. Установка
gem install mega_matrix-0.1.0.gem
🧪 Использование
require 'mega_matrix'

# Создание матриц
m1 = MegaMatrix::Matrix.new([[1, 2], [3, 4]])
m2 = MegaMatrix::Matrix.new([[5, 6], [7, 8]])

# Сложение
puts m1 + m2

# Умножение на скаляр
result = MegaMatrix::Arifmetrix.multi_scalar(m1.data, 2)
puts result

# Доступ к элементам
value = MegaMatrix::Accesstrix.get(m1.data, 0, 1)
puts value
```
Аналогично сложению можно использовать операции умножения на скаляр (*), возведение в степень (**), вычитание (-), деление на скаляр (/).
Присутствует доступ к элементам с их последующим изменением с помощью привычной многим индексации ([])
Сложные операции типа Адамарово произведение, изменение порядка столбцов или строк, транспонирование, решейп и т.п. вводятся через конструкцию "элемент.операция(аргумент, если нужен)"

🧠 Архитектура проекта

Проект разделён на модули:

* MegaMatrix::Matrix — основной класс матрицы

* Genetrix — генерация матриц
  Создение нулевой, единичной, случайной, по сложному условию, диагональной, Гильбертовой матриц. Оформлен вывод в поток.

* Arifmetrix — арифметические операции
  Классические арифметические операции: сложение, вычитание, умножение, деление на скаляр, возведение в степень (нулевая степень понимается, как желание переделать матрицу в единичную!!!)

* Accesstrix — доступ к элементам
  Доступ к элементам матрицы и их изменение, работа со строками, столбцами, а именно: вставка, удаление, обмен.

* Linetrix — линейная алгебра (расширение)
  Классическая линейная алгебра: ранг, определитель, след, транспонирование, взятие обратной (выполнено отдельно от степенной операции из Arifmetrix),

* Spectrix — дополнительные операции
  Особые операции, такие как изменение структуры матрицы без изменения данных, присоединение других матриц к исходной и т.п.
